# Submission Validation System (AWS)

## Overview
This project is a serverless system that validates student submissions automatically.

## Architecture
S3 → Lambda → DynamoDB → SNS

## Features
- Checks missing fields
- Detects late submissions
- Stores results
- Sends email notifications

## Technologies
- AWS S3
- AWS Lambda
- DynamoDB
- SNS
